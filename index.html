Бакалаврська робота

Тема: Веб-застосунок інтернет-магазину квітів

Сумський державний університет

Автор: Костенко Юлія

Опис роботи:

Розробка веб-застосунку на базі CMS WordPress для інтернет-магазину квітів. Проєкт передбачає створення адаптивного веб-сайту для продажу квітів, що включає каталог товарів, кошик замовлень, інтеграцію з платіжними системами та функціонал для управління замовленнями. Вибір CMS WordPress обумовлений її гнучкістю, широким вибором плагінів та можливістю кастомізації під потреби конкретного бізнесу.

Актуальність теми:

Онлайн-продажі квітів набувають популярності, оскільки клієнти цінують можливість швидко обрати та замовити букет із доставкою. Використання WordPress дозволяє створювати масштабовані веб-магазини з інтуїтивно зрозумілим інтерфейсом як для клієнтів, так і для адміністраторів сайту. Автоматизація процесу обробки замовлень, інтеграція з системами оплати та гнучкі налаштування дозволяють зменшити витрати на ведення бізнесу.

Мета розробки:

Метою проєкту є створення веб-застосунку для продажу квітів, який забезпечить зручний інтерфейс, швидку обробку замовлень і можливість інтеграції з популярними сервісами доставки та оплати.

Основні завдання:

Дослідити особливості онлайн-продажу квітів та потреби цільової аудиторії.

Розробити інтуїтивно зрозумілий користувацький інтерфейс.

Оптимізувати продуктивність сайту для швидкого завантаження сторінок.

Інтегрувати сучасні платіжні системи для безпечних онлайн-оплат.

Забезпечити мобільну адаптацію сайту для зручності на смартфонах.

Методологія розробки:

Для створення веб-застосунку використовується CMS WordPress з кастомізацією теми та плагінів під потреби магазину. В якості шаблонної системи використовується Elementor, що дозволяє швидко змінювати контент.

Очікувані результати:

Фінальним продуктом стане функціональний інтернет-магазин квітів, що працює на WordPress, з інтегрованими модулями для керування замовленнями, онлайн-оплатою та доставкою. Веб-застосунок забезпечить автоматизацію процесу замовлення квітів, зменшить час оформлення покупки та покращить клієнтський досвід. Користувачі зможуть швидко знаходити необхідні товари, оформлювати замовлення та оплачувати покупки, а адміністратори – зручно керувати контентом магазину.
